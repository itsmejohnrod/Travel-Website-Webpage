# Travel Website Webpage

A beautiful travel website showcasing destinations in the Philippines.

## Features

- Modern, responsive design
- Hero section with search functionality
- Destination showcase (Baguio, Coron, Bohol, Boracay)
- Smooth scrolling navigation
- Interactive UI elements

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

Simply open `index.html` in your web browser to view the website.
